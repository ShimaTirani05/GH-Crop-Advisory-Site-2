🍈 Panduan Install GH Crop Advisory ke HP

File ini menjelaskan cara menginstall aplikasi Greenhouse Crop Advisory ke HP Android dan iPhone/iPad tanpa App Store.

✅ Kabar baik: Aplikasi ini adalah PWA (Progressive Web App) — bisa diinstall langsung dari browser, GRATIS, tanpa perlu Google Play Store atau App Store.

LANGKAH 1 — Upload File ke Hosting

Sebelum bisa diinstall di HP, file aplikasi harus diunggah ke internet (hosting). Pilih salah satu cara di bawah:

A Netlify GRATIS Android iOS — Paling mudah, direkomendasikan
  1. Buka netlify.com → klik Sign up (gratis dengan email atau GitHub)
  2. Setelah login, di halaman utama ada kotak besar: "Drag and drop your site folder here"
  3. Buka folder sadaya-deploy di komputer Anda
  4. Drag seluruh folder dan drop ke kotak Netlify tersebut
  5. Tunggu 30–60 detik → Netlify memberi URL seperti: https://random-name-xyz.netlify.app
  6. URL ini adalah alamat aplikasi Anda — bagikan ke seluruh tim!
B GitHub Pages GRATIS — Cocok jika sudah punya akun GitHub
  1. Buat akun di github.com
  2. Klik New repository → nama: gh-crop-advisory → Public → Create
  3. Upload semua file dari folder sadaya-deploy (klik "uploading an existing file")
  4. Masuk ke Settings → Pages → Source: Deploy from branch → main → Save
  5. URL: https://[username].github.io/gh-crop-advisory
C cPanel / Shared Hosting (jika sudah punya domain & hosting)
  1. Login ke cPanel → File Manager
  2. Masuk ke folder public_html
  3. Upload semua file dan folder dari sadaya-deploy
  4. Pastikan hosting mendukung HTTPS (SSL aktif) — wajib untuk PWA
  5. Akses di: https://domain-anda.com

LANGKAH 2 — Install di HP Android 📱

1 Buka Chrome di HP Android (bukan Samsung Browser atau Firefox)
2 Ketik URL aplikasi Anda (dari Netlify/GitHub Pages) di address bar → Enter
3 Tunggu 5–10 detik → akan muncul banner di bawah layar: "Tambahkan GH Crop Advisory ke layar utama"
4 Ketuk Install atau Tambahkan → Konfirmasi → selesai!
⚠️ Jika banner tidak muncul otomatis:
Ketuk ⋮ (tiga titik) pojok kanan atas Chrome → pilih "Tambahkan ke layar utama" atau "Install app"

LANGKAH 3 — Install di iPhone / iPad 🍎

⚠️ Wajib gunakan Safari! Chrome di iPhone tidak mendukung install PWA.
1 Buka Safari di iPhone/iPad (bukan Chrome atau aplikasi browser lain)
2 Ketik URL aplikasi di address bar → Enter
3 Ketuk ikon Bagikan (kotak dengan panah ke atas ↑) — ada di bagian tengah bawah layar
4 Scroll ke bawah di menu yang muncul → pilih "Tambahkan ke Layar Utama" (Add to Home Screen)
5 Konfirmasi nama "GH Crop Advisory" → Ketuk Tambahkan → selesai!

LANGKAH 4 — Bagikan URL ke Seluruh Tim

Setelah hosting aktif, bagikan URL ke semua anggota tim melalui WhatsApp. Setiap orang bisa install sendiri di HP masing-masing dengan langkah di atas.

📲 Contoh pesan WhatsApp:

"Halo tim, silakan install aplikasi Greenhouse Crop Advisory di HP masing-masing:
👉 [URL NETLIFY ANDA]
Android: buka di Chrome → ketuk Install
iPhone: buka di Safari → Bagikan → Tambahkan ke Layar Utama"
